### Overview

A small full-stack project built for a coding challenge. The stack combines <Anchor link="https://nestjs.com/" text="NestJS" variant="link" isExternal /> on the backend with <Anchor link="https://nextjs.org/" text="Next.js" variant="link" isExternal /> on the frontend, with PostgreSQL serving as the database.

It was mainly an exercise in getting familiar with NestJS and the App Router in Next.js, while building a simple product-style interface for managing coffee items.

### Key Features

1. **Automatic data seeding**  
  The backend seeds coffee items on startup, so the app is usable immediately.
2. **Create flow via modal**  
  New coffee items are created through a modal form.
  Data is sent to the backend and persisted in the database.
3. **UI animations**  
  Navbar reacts to scroll
  Modal includes entry/exit animations
4. **Filtering**  
  Filter coffee items by type (Robusta or Arabica)
